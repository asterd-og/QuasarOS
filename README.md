# QuasarOS
x86_64 Hobby Operating System

# Features
- [x] GDT
- [x] IDT
- [x] PMM
- [x] VMM
- [x] Heap
- [x] Serial
- [x] Printf
- [x] PS/2 Keyboard
- [x] PS/2 Mouse
- [x] Multi-Tasking
- [x] ELF Loading
- [x] Syscalls
- [x] Ramdisk
- [x] Simple IPC signal system
- [ ] QuasarOS Cross Compiler
